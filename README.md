# Requirements

- fish shell (3.0.0 <=)
- homebrew

# Installation

```sh
% git clone git://github.com/unlovingly/dotfiles.git ~/.files
```

```sh
% cd ~/.files
```

```sh
% brew bundle
% task install
```

# Environment Variables

## `XDG_CONFIG_HOME`

```fish
set -g -x XDG_CONFIG_HOME ~/.config
```
